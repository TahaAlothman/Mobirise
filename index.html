How website builder software works

Drag n drop blocks

Start with creating a new website and picking up the template. Then expand the blocks panel with the big red "plus" button in the lower right corner and start dragging the blocks you like.

Edit and style inline

Edit the content of each block just like you would in a regular text editor, click on media elements to replace it by your own image, video or icon.

Set block
parameters

Set the appropriate options through the Parameters Panel being brought out by the blue "gear" button.

Preview and publish

Click on "Mobile View" in the website maker to check how your site looks on different devices and publish to local drive, FTP or Mobirise subdomain.
